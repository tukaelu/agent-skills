# アスキーアート例

各図種のアスキーアートスケッチの参考例。実際の内容に合わせてノード名・ラベルを書き換えて使う。

## フローチャート
```
[開始]
  |
  v
[ログイン画面] --認証失敗--> [エラー表示]
  |                               |
  | 認証成功                    再試行
  v                               |
[ダッシュボード]  <--------------/
```

## シーケンス図
```
ユーザー     フロント    バックエンド    DB
  |             |             |          |
  |--ログイン-->|             |          |
  |             |---認証要求->|          |
  |             |             |--照会--> |
  |             |             |<--結果---|
  |<--結果------|<--応答------|          |
```

## 状態遷移図
```
[待機中] --注文受付--> [処理中] --発送完了--> [配送中] --受取確認--> [完了]
                          |
                       キャンセル
                          |
                          v
                       [取消済]
```

## クラス図
```
[Animal]
 # name: String
 # age: int
 + speak(): void
      △
      |（継承）
  ----+----
  |        |
[Dog]    [Cat]
 + breed   + indoor: bool
 + fetch() + purr()
```

## ER図
```
[User] ---1対多--- [Order] ---1対多--- [OrderItem] ---多対1--- [Product]
 id(PK)             id(PK)              id(PK)                  id(PK)
 name               user_id(FK)         order_id(FK)            name
 email              status              product_id(FK)          price
                                        quantity
```
