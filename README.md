# Agent Skills

コーディングエージェント向けのスキルコレクション

## Installation

```bash
npx skills add tukaelu/agent-skills
```

## Skills

### committer

変更内容を解析し、適切な粒度でコミットをグループ化して作成する。

### git-branch-cleaner

マージ済みブランチをローカル・リモートから安全に削除する。

### interest-discovery

はてなブックマークから、興味プロフィールに基づいて関連コンテンツを発見する。

### mac-maintainer

Homebrew・anyenv 経由でMac環境のパッケージを一括アップデートする。

### proofreader

日本語・英語の校正・推敲を行い、誤字脱字・文法ミス・表現の改善点を提示する。

### researcher

テーマの調査・分析を通じて意思決定に役立つ知見をレポートや示唆として提供する。

### skill-composer

スキルの新規作成・既存スキルのメンテナンスを一貫してサポートする。

### skill-reviewer

Claude Code スキルをベストプラクティスに基づいてレビューし、改善案を提示する。

### text-explainer

難しい表現・専門用語・回りくどい言い回しをわかりやすい日本語に書き直す。
